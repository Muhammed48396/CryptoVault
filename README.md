# 🔐 CryptoVault

CryptoVault, Python ile geliştirilmiş bir terminal tabanlı güvenlik aracıdır.

## Özellikler

- 🔒 Metin Şifreleme
- 🔓 Metin Çözme
- 🔑 Güçlü Parola Üretici
- 🛡️ Security Tools
- 🎨 Renkli Terminal Arayüzü

## Kurulum

```bash
pkg update
pkg install python git
pip install cryptography
pip install colorama

git clone https://github.com/muhammedyapar13-collab/CryptoVault.git

cd CryptoVault

python main.py
```

# 🔐 CryptoVault

CryptoVault, Python ile geliştirilmiş bir terminal tabanlı güvenlik aracıdır.

## Özellikler

- 🔒 Metin Şifreleme
- 🔓 Metin Çözme
- 🔑 Güçlü Parola Üretici
- 🛡️ Security Tools
- 🎨 Renkli Terminal Arayüzü

## Kurulum

```bash
pkg update
pkg install git python -y

git clone https://github.com/Muhammed48396/CryptoVault.git
cd CryptoVault

pip install -r requirements.txt

python main.py
```
